# Implementation Gap Matrix

Status meanings:

- `Implemented`: code exists and the repo contains direct evidence plus a practical validation path.
- `Partial`: code exists, but the control is incomplete, inconsistent, or bypassed by another active path.
- `Documented Only`: expectations exist in docs, but repo evidence is not strong enough to mark implementation complete.
- `Missing`: no credible implementation evidence found in the current repo.
