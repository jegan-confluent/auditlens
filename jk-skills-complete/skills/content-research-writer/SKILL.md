# Content Research & Writing

## Research Process
1. Define topic and target audience
2. Identify key questions to answer
3. Gather sources (5-10 quality sources)
4. Extract key points and quotes
5. Create outline
6. Write draft
7. Edit and refine

## Content Structure
```markdown
# Title (60 chars max for SEO)

## Introduction (hook + thesis)
- Hook: Start with compelling stat, question, or story
- Context: Why this matters
- Thesis: What reader will learn

## Main Section 1
### Subpoint A
### Subpoint B

## Main Section 2
### Subpoint A
### Subpoint B

## Conclusion
- Summary of key points
- Call to action
```

## SEO Checklist
- [ ] Keyword in title
- [ ] Keyword in first 100 words
- [ ] Keyword in at least one heading
- [ ] Meta description (155 chars)
- [ ] Internal links
- [ ] External links to authority sources
- [ ] Image alt text

## Writing Tips
- Use active voice
- Keep sentences short (15-20 words avg)
- Use transition words
- Break up text with subheadings
- Include examples and data
